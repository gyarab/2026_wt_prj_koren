# Databáze aut

Tento projekt je webová aplikace vytvořená pomocí frameworku Django, která bude fungovat jako databáze aut. 
Cílem aplikace je ukládat a zobrazovat informace o různých automobilech.

Každé auto v databázi bude obsahovat základní informace, například značku, model, rok výroby, typ motoru, výkon a cenu. 
Uživatelé budou moci procházet seznam aut a zobrazit si detailní informace o jednotlivých vozidlech.

Aplikace bude využívat Django modely pro ukládání dat do databáze a Django šablony pro zobrazení informací na webových stránkách. 
V budoucnu by bylo možné přidat například vyhledávání aut podle značky, filtrování podle roku výroby nebo přidávání nových aut pomocí administračního rozhraní Django.
